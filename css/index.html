CSS 레퍼런스 북

CSS 레퍼런스 속성과 설명을 나타내는 표입니다.
속성 설명
align-content align-content 속성은 콘텐츠 사이와 콘텐츠 주위 빈 공간을 플렉스박스의 교차축, 그리드의 블록 축을 따라 배치하는 방식을 결정합니다.
align-items align-items 교차축에 대해 flex 항목들을 정렬하는 방식을 제어. flex 컨테이너에 지정하는 속성.
align-self align-self
all all 단축 속성은 요소의 unicode-bidi (en-US), direction (en-US), CSS 사용자 지정 속성을 제외한 모든 속성을 초기화합니다.
animation animation 속성은 다수의 스타일을 전환하는 애니메이션을 적용합니다
animation-delay animation-delay 속성은 애니메이션이 시작할 시점을 지정합니다.
animation-direction animation-direction 속성은 애니메이션이 앞으로, 뒤로 또는 앞뒤로 번갈아 재생되어야하는지 여부를 지정합니다.
animation-duration animation-duration속성은 애니메이션이 한 사이클을 완료하는 데 걸리는 시간을 지정합니다.
animation-fill-mode animation-fill-mode속성은 CSS 애니메이션이 실행 전과 후에 대상에 스타일을 적용하는 방법을 지정합니다.
animation-iteration-count animation-iteration-count속성은 애니메이션 순서가 중지되기 전에 재생되어야 하는 횟수를 설정합니다.
animation-name animation-name속성은 요소에 적용할 애니메이션을 설명하는 하나 이상의 @keyframes at-rules의 이름을 지정합니다.
animation-play-state animation-play-state속성은 애니메이션이 실행 중인지 일시 중지되었는지 설정합니다.
animation-timing-function animation-timing-function속성은 각 주기의 지속 시간 동안 애니메이션이 진행되는 방식을 설정합니다.
backface-visibility backface-visibility속성은 사용자를 향할 때 요소의 뒷면을 볼 수 있는지 여부를 설정합니다.
background background속성은 색상, 이미지, 원점 및 크기 또는 반복 방법과 같은 모든 배경 스타일 속성을 한 번에 설정합니다.
background-attachment background-attachment속성은 배경 이미지의 위치가 뷰포트 내에서 고정되는지 또는 포함하는 블록과 함께 스크롤되는지를 설정합니다.
background-blend-mode background-blend-mode속성은 요소의 배경 이미지가 서로 그리고 요소의 배경색과 어떻게 혼합되어야 하는지를 설정합니다.
background-clip background-clip속성은 요소의 배경이 테두리 상자, 패딩 상자 또는 콘텐츠 상자 아래로 확장되는지 여부를 설정합니다.
background-color background-color속성은 요소의 배경색을 설정합니다.
background-image background-image속성은 요소에 하나 이상의 배경 이미지를 설정합니다.
background-origin background-origin속성은 배경의 원점을 설정합니다: 경계 시작부터, 경계 내부 또는 패딩 내부.
background-position background-position속성은 각 배경 이미지의 초기 위치를 설정합니다. 위치는 background-origin에 의해 설정된 위치 레이어에 상대적입니다.
background-repeat background-repeat속성은 배경 이미지가 반복되는 방식을 설정합니다. 배경 이미지는 가로 및 세로 축을 따라 반복되거나 전혀 반복되지 않을 수 있습니다.
background-size background-size속성은 요소의 배경 이미지 크기를 설정합니다. 이미지는 사용 가능한 공간에 맞게 원래 크기로 유지하거나 늘리거나 제한할 수 있습니다.
border border속성은 요소의 테두리를 설정합니다. border-width, border-style, border-color 값을 설정합니다.
border-bottom border-bottom속성은 요소의 아래쪽 테두리를 설정합니다. border-bottom-width, border-bottom-style 및 border-bottom-color 값을 설정합니다.
border-bottom-color >border-bottom-color속성은 요소의 하단 테두리 색상을 설정합니다. CSS 약식 속성 border-color 또는 border-bottom으로 설정할 수도 있습니다.
border-bottom-left-radius border-bottom-left-radius속성은 모서리의 곡률을 정의하는 타원의 반경(또는 semi-major 및 semi-minor 축의 반경)을 지정하여 요소의 왼쪽 하단 모서리를 반올림합니다.
border-bottom-right-radius border-bottom-right-radius 속성은 모서리의 곡률을 정의하는 타원의 반지름(또는 반장축 및 반단축의 반지름)을 지정하여 요소의 오른쪽 아래 모서리를 반올림합니다.
border-bottom-style border-bottom-style속성은 요소 하단 테두리의 선 스타일을 설정합니다.
border-bottom-width border-bottom-width속성은 요소의 아래쪽 테두리 너비를 설정합니다.
border-collapse border-collapse속성은 내부의 셀이 공유 또는 별도의 테두리를 가질지 여부를 설정합니다.
border-color border-color속성은 요소의 테두리 색상을 설정합니다.
border-image border-image속성은 주어진 요소 주위에 이미지를 그립니다. 요소의 일반 테두리를 대체합니다.
border-image-outset border-image-outset속성은 요소의 테두리 이미지가 테두리 상자에서 설정되는 거리를 설정합니다.
border-image-repeat border-image-repeat속성은 소스 이미지의 가장자리 영역이 요소의 테두리 이미지 크기에 맞게 조정되는 방법을 정의합니다.
border-image-slice border-image-slice속성은 border-image-source로 지정된 이미지를 영역으로 나눕니다. 이러한 영역은 요소의 테두리 이미지의 구성 요소를 형성합니다.
border-image-source border-image-source속성은 요소의 테두리 이미지를 만드는 데 사용되는 소스 이미지를 설정합니다.
