MYSQL

MYSQL은 데이터베이스 소프트웨어입니다 일반덕으로 데이터를 추구하거나 검색, 추출하는 기능을 모두 포함하여 테이터 베이스라고 부릅니다.

MYSQL은 세계에서 가장 많이 쓰니는 오픈소스의 관계형 데이터베이스 관리 시스템(RDBMS)입니다. MYSQL은 PHP 스크립트 언어화 상호 연동이 잘 되면서 오프소스로 개발된 무료 프로그랩 입니다. 그래서 홈페이지나 쇼핑물(워드프로세스,CAFE24,제로보드,그누보드)등 일반적으로 웹 개발에 널리 사용하고 있습니다
