CSS 단위

CSS에서 사용하는 단위는 font-size, text-indent, line-height, width, height, padding 등 크기를 나타내는 값에 쓰인다. 크게는 절대 크기가 변하지 않는 절대 단위와 상황에 따라 크기가 변하는 상대 단위로 나뉩니다

rem

rem은 최상위 엘리먼트의 폰트 사이즈를 기준으로 한 단위입니다


<body>
    <div class="test" <Test </div >
    </body>
body {
    font-size: 14px;
}
div {
    font-size: 1.2em; // calculated at 14px * 1.2, or 16.8px
}
