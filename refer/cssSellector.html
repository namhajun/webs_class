클래스 선택자 .class {color:#fff} class 클래스를 아이디를 선택하여 글씨랙을 하얀색으로 변경합니다

속성 선택자

속성 예시 설명
하위 선택자
(Descendant Selector)		 div p {color:#fff} 후손이라는 말은 특정 태그에 종속되어 있는 모든 하위 요소들을 의미합니다. 위 구조에서 html 태그의 후손에 해당하는 태그들인 <head><body><h1><div> 태그들에 이에 해당합니다.
자식 선택자
(Child selector)		 div >p {color:#fff} 첫 번째 선택자와 일치하는 요소의 직계 자식인 두 번째 선택자와 일치하는 요소만 일치합니다.
형제 선택자
(Sibling Selector) 		div~ p {color:#fff} 일반 형제 선택자는 해당 요소와 형제 관계에 있으며, 해당 요소보다 뒤에 존재하는 특정 타입의 요소를 모두 선택한다.
인접 형제 선택자(Adjacent Sibling Selector) div+ p {color:#fff} 인접 형제 선택자는 해당 요소와 형제 관계에 있으며, 해당 요소의 바로 뒤에 존재하는 특정 타입의 요소를 모두 선택한다.

가상 선택자

속성 예시 설명
:first-child li:first-child{color: #fff;} 형제 요소 그룹 중 첫 번째 요소를 나타냅니다.
:first-of-type li:first-child{color: #fff;}
:last-of-type p:first-child{color: #fff;} 형제 요소 그룹 중 해당 유형의 첫 번째 요소를 나타냅니다.
:nth-child() p:first-child{color: #fff;} \형제 그룹 간의 위치에 따라 요소를 찾습니다.
:nth-last-child() li:first-child{color: #fff;} 끝에서 계산하여 형제 그룹 간의 위치를 ​​기반으로 요소를 찾습니다.
:nth-of-type() li:first-child{color: #fff;} 같은 유형(태그 이름)의 형제 간의 위치를 ​​기반으로 요소를 찾습니다.
:nth-last-of-type() p:first-child{color: #fff;} 끝부터 계산하여 동일한 유형(태그 이름)의 형제 간의 위치를 ​​기반으로 요소를 찾습니다.
:only-child p:first-child{color: #fff;} 형제가 없는 요소를 나타냅니다.
:only-of-type() p:first-child{color: #fff;} 동일한 유형의 형제가 없는 요소를 나타냅니다.
:not() li:not(:nyh-child(2){color: #fff;} 선택한 요소 중에서 전달받은 선택자에 해당하거나, 함수 호출의 결과가 참(true)인 요소를 제외한 나머지 요소를 모두 선택함.
:root root{color: #fff;} 문서를 나타내는 트리의 루트 요소와 일치합니다.
:empty empty{color: #fff;} 빈 문은 문을 제공하지 않는 데 사용되지만 JavaScript 구문은 문을 예상합니다
