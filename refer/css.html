animation animation 속성은 다수의 스타일을 전환하는 애니메이션을 적용합니다
animation-delay animation-delay 속성은 애니메이션이 시작할 시점을 지정합니다.
animation-direction animation-direction 속성은 애니메이션이 앞으로, 뒤로 또는 앞뒤로 번갈아 재생되어야하는지 여부를 지정합니다.
animation-duration animation-duration속성은 애니메이션이 한 사이클을 완료하는 데 걸리는 시간을 지정합니다.
animation-fill-mode animation-fill-mode속성은 CSS 애니메이션이 실행 전과 후에 대상에 스타일을 적용하는 방법을 지정합니다.
animation-iteration-count animation-iteration-count속성은 애니메이션 순서가 중지되기 전에 재생되어야 하는 횟수를 설정합니다.
animation-name animation-name속성은 요소에 적용할 애니메이션을 설명하는 하나 이상의 @keyframes at-rules의 이름을 지정합니다.
animation-play-state animation-play-state속성은 애니메이션이 실행 중인지 일시 중지되었는지 설정합니다.
animation-timing-function animation-timing-function속성은 각 주기의 지속 시간 동안 애니메이션이 진행되는 방식을 설정합니다.
backface-visibility backface-visibility속성은 사용자를 향할 때 요소의 뒷면을 볼 수 있는지 여부를 설정합니다.
background background속성은 백그라운드 속성을 일괄적으로 설정합니다
background-attachment background-attachment속성은 배경 이미지의 고정 여부를 설정합니다
background-blend-mode background-blend-mode속성은 요소의 배경 혼합했을때의 상태를 설정합니다.
background-clip background-clip속성은 백그라운드 이미지의 위치 기준점을 설정합니다
background-color background-color속성은 백그라운드 색을 설정합니다.
background-image background-image속성은 배경 이미지를 및 배경 속성을 설정합니다.
background-origin background-origin속성은 백그라운드 이미지의 위치 기준점을 설정합니다
background-position background-position속성은 백그라운드 이미지의 위치 영역을 설정합니다.
background-repeat background-repeat속성은 배경 이미지의 반복 여부를 설정합니다
background-size background-size속성은 백그라운드 이미지의 사이즈를 설정합니다.
border border속성은 테두리속성을 일괄적으로설정합니다.
border-bottom border-bottom속성은 아래쪽 속성을 일괄적으로 설정합니다
border-bottom-color >border-bottom-color속성은 테두리 아래쪽 색 속성을 설정합니다.
border-bottom-left-radius border-bottom-left-radius속성은 아래부분 왼쪽 모서리 굴곡을 설정합니다.
border-bottom-right-radius border-bottom-right-radius 속성은 아래부분 오른쪽 모서리 굴곡을 설정합니다
border-bottom-style border-bottom-style속성은 테두리 아래쪽 스타일 속성을 설정합니다
border-bottom-width border-bottom-width속성은 테두리 아래쪽 두께 속성을 설정합니다
border-collapse border-collapse속성은 테이블 테두리 분리 여부를 설정합니다
border-color border-color속성은 테두리 색속성을 설정합니다.
border-image border-image속성은 주어진 요소 주위에 이미지를 그립니다. 요소의 일반 테두리를 대체합니다.
border-image-outset border-image-outset속성은 요소의 테두리 이미지가 테두리 상자에서 설정되는 거리를 설정합니다.
border-image-repeat border-image-repeat속성은 소스 이미지의 가장자리 영역이 요소의 테두리 이미지 크기에 맞게 조정되는 방법을 정의합니다.
border-image-slice border-image-slice속성은 border-image-source로 지정된 이미지를 영역으로 나눕니다. 이러한 영역은 요소의 테두리 이미지의 구성 요소를 형성합니다.
border-image-source border-image-source속성은 요소의 테두리 이미지를 만드는 데 사용되는 소스 이미지를 설정합니다.
border-image-width border-image-width속성은 요소의 테두리 이미지 너비를 설정합니다.
border-left border-left속성은 요소의 왼쪽 테두리에 대한 모든 속성을 설정합니다.
border-left-color border-left-color 속성은 요소의 왼쪽 테두리 색상을 설정합니다. CSS 약식 속성 border-color 또는 border-left로 설정할 수도 있습니다.
border-left-style border-left-style속성은 요소의 왼쪽 테두리 선 스타일을 설정합니다.
border-left-width border-left-width속성은 요소의 왼쪽 테두리 너비를 설정합니다.
border-radius border-radius속성은 요소의 바깥쪽 테두리 모서리를 둥글게 만듭니다. 단일 반경을 설정하여 원형 모서리를 만들거나 두 개의 반경을 설정하여 타원형 모서리를 만들 수 있습니다.
border-right border-right속성은 요소 오른쪽 테두리의 모든 속성을 설정합니다.
border-right-color border-right-color속성은 요소의 오른쪽 테두리 색상을 설정합니다. CSS 약식 속성 border-color 또는 border-right로 설정할 수도 있습니다.
border-right-style border-right-style속성은 요소의 오른쪽 테두리 선 스타일을 설정합니다.
border-right-width border-right-width속성은 요소의 오른쪽 테두리 너비를 설정합니다.
border-spacing border-spacing속성은 인접한 <table>셀의 경계 사이의 거리를 설정합니다. 이 속성은 border-collapse가 분리된 경우에만 적용됩니다.
border-style border-style 속성은 요소 테두리의 네 면 모두에 대한 선 스타일을 설정합니다.
border-top border-top속성은 요소 위쪽 테두리의 모든 속성을 설정합니다.
border-top-color border-top-color속성은 요소의 상단 테두리 색상을 설정합니다. CSS 약식 속성 border-color 또는 border-top을 사용하여 설정할 수도 있습니다.
border-top-left-radius border-top-left-radius속성은 모서리의 곡률을 정의하는 타원의 반경(또는 semi-major 및 semi-minor 축의 반경)을 지정하여 요소의 왼쪽 상단 모서리를 반올림합니다.
border-top-right-radius
border-top-style border-top-left-radius 속성은 모서리의 곡률을 정의하는 타원의 반경(또는 semi-major 및 semi-minor 축의 반경)을 지정하여 요소의 왼쪽 상단 모서리를 반올림합니다.
border-top-width border-top-width속성은 요소의 위쪽 테두리 너비를 설정합니다.
border-width border-width속성은 요소의 테두리 너비를 설정합니다.
bottom bottom속성은 배치된 요소의 수직 위치 설정에 참여합니다. 위치가 지정되지 않은 요소에는 영향을 주지 않습니다.
box-decoration-break box-decoration-break속성은 여러 줄, 열 또는 페이지에 걸쳐 분할될 때 요소의 조각을 렌더링하는 방법을 지정합니다.
box-shadow box-shadow속성은 요소의 프레임 주위에 그림자 효과를 추가합니다. 여러 효과를 쉼표로 구분하여 설정할 수 있습니다. 상자 그림자는 요소에 상대적인 X 및 Y 오프셋, 흐림 및 확산 반경, 색상으로 설명됩니다.
box-sizing box-sizing속성은 요소의 전체 너비와 높이를 계산하는 방법을 설정합니다.
caption-side caption-side속성은 지정된 쪽에 테이블의 <caption> 콘텐츠를 배치합니다. 값은 테이블의 쓰기 모드에 상대적입니다.
caret-color caret-color속성은 삽입 캐럿의 색상을 설정하며, 입력된 다음 문자가 삽입될 표시되는 마커입니다. 이를 텍스트 입력 커서라고도 합니다. 캐럿은 <input>과 같은 요소나 contenteditable 속성이 있는 요소에 나타납니다. 캐럿은 일반적으로 눈에 잘 띄도록 깜박이는 가는 세로선입니다. 기본적으로 검은색이지만 이 속성으로 색상을 변경할 수 있습니다.
clear clear속성은 요소가 선행하는 부동 요소 아래로 이동(선택 취소)되어야 하는지 여부를 설정합니다. clear 속성은 부동 요소와 부동 요소가 아닌 요소에 적용됩니다.
clip clip속성은 요소의 보이는 부분을 정의합니다. clip 속성은 절대 위치 요소, 즉 position:absolute 또는 position:fixed 요소에만 적용됩니다.
color color속성은 요소의 텍스트 및 텍스트 장식의 전경색 값을 설정하고 <currentcolor> 값을 설정합니다. currentcolor는 다른 속성의 간접 값으로 사용될 수 있으며 border-color와 같은 다른 색상 속성의 기본값입니다.
column-count column-count속성은 요소의 내용을 지정된 수의 열로 나눕니다.
column-fill column-fill속성은 열로 나눌 때 요소의 내용이 균형을 이루는 방법을 제어합니다.
column-gap column-gap속성은 요소의 열 사이의 간격(거터) 크기를 설정합니다.
column-rule column-rule속성은 다중 열 레이아웃에서 열 사이에 그려지는 선의 너비, 스타일 및 색상을 설정합니다.
column-rule-color column-rule-color속성은 다중 열 레이아웃에서 열 사이에 그려지는 선의 색상을 설정합니다.
column-rule-style column-rule-style속성은 다중 열 레이아웃에서 열 사이에 그려지는 선의 스타일을 설정합니다.
column-rule-width column-rule-width속성은 다중 열 레이아웃에서 열 사이에 그려지는 선의 너비를 설정합니다.
