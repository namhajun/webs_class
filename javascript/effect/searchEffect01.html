indexOf( )를 이용하여 CSS 속성 검색하기

CSS 속성 검색하기

  • all : all 속성은 CSS 속성을 재설정하는데 사용할 수 있는 약식 속성입니다.
  • animation : animation 속성은 애니메이션 속성을 설정하기 위한 약식 속성입니다.
  • animation-delay : animation-delay 속성은 애니메이션이 시작되는시간을 설정합니다.
  • animation-direction : animation-direction 속성은애니메이션이 움직이는 방향을 설정합니다.
  • animation-duration : animation-duration 속성은애니메이션이 움직이는 시간을 설정합니다.
  • animation-fill-mode : animation-fill-mode 속성은애니메이션이 끝난 후의 상태를 설정합니다.
  • animation-iteration-count : animation-iteration-count 속성은 애니메이션 반복 횟수를 설정합니다.
  • animation-name : animation-name 속성은 애니메이션 키프레임 이름을 설정합니다.
  • animation-play-state : animation-play-state속성은 애니메이션의 진행상태를 설정합니다.
  • backdrop-filter : backdrop-filter 속성은 요소 뒤에 필터효과를설정합니다.
  • backface-visibility : backface-visibility 속성은 요소 뒷면 출력 여부 설정합니다.
  • background :background 속성은 대부분의 배경 속성을 스타일 시트의 같은 위치에 설정하기 위한 약식 속성입니다.
  • background-attachment : background-attachment 속성은 배경 이미지가 뷰포트에 상대적으로 고정되어 있는지 또는 요소 또는 해당 콘텐츠와 함께 스크롤되는지 여부를 지정하는 데 사용됩니다.
  • background-blend-mode : background-blend-mode 속성은 요소의 각 배경 레이어에 대한 혼합 모드를 지정하는 데 사용됩니다.
  • background-clip : background-clip 속성은 배경이 칠해지는 영역을 지정합니다.
  • background-color : background-color 속성은 요소의 배경색을 설정하는 데 사용됩니다
  • background-image : background-image 속성은 요소의 배경 이미지를 설정합니다.
  • background-origin : background-origin 속성은 배경 위치 지정 영역이라고 하는 것을 지정합니다
  • background-position : background-position 속성은 하나 이상의 쉼표로 구분된 position 값을 취합니다
  • background-repeatbackground-repeat : background-repeat 속성은 이미지를 반복(타일링)할지 여부와 방법을 정의합니다.
  • border-image-slice : border-image-slice 속성은 테두리 이미지로 사용할 이미지를 "슬라이스"하는 데 사용됩니다.
  • background-size : background-size 속성은 배경 이미지의 크기를 지정하는 데 사용됩니다.
  • border : border 속성은 상자의 상하좌우 테두리의 너비, 스타일, 색상을 지정하는 약식 속성입니다.
  • border-bottom : border-bottom 속성은 요소 하단 테두리의 너비, 스타일 및 색상을 지정하는 데 사용되는 약식 속성입니다
  • border-bottom-color : border-bottom-color 속성은 요소의 아래쪽 테두리 색상을 설정하는 데 사용됩니다.
  • border-bottom-left-radius : border-bottom-left-radius 속성은 요소의 왼쪽 하단 모서리를 둥글게 만드는 데 사용됩니다.
  • border-top-right-radius : border-top-right-radius 속성은 요소의 오른쪽 상단 모서리를 둥글게 만드는 데 사용됩니다.
  • border-top-style : background-repeat 속성은 이미지를 반복(타일링)할지 여부와 방법을 정의합니다.
  • border-top-width :border-top-width 속성은 요소의 위쪽 테두리 너비를 설정하는 데 사용됩니다.
  • border-left : border-left 속성은 요소의 왼쪽 테두리의 너비, 스타일 및 색상을 지정하는 데 사용되는 약식 속성입니다.
  • border-left-color : border-left-color 속성은 요소의 왼쪽 테두리 색상을 설정하는 데 사용됩니다.
  • border-left-style : border-left-style 속성은 테두리 이미지가 설정되어 있지 않는 한 요소의 왼쪽 테두리 선 스타일(실선, 이중, 대시 등)을 지정하는 데 사용됩니다.
  • border-left-width : border-left-width 속성은 요소의 왼쪽 테두리 너비를 설정하는 데 사용됩니다.
  • border-radius : border-radius 속성은 요소의 모서리를 둥글게 만드는 데 사용됩니다.
  • border-right : border-right 속성은 요소 오른쪽 테두리의 너비, 스타일 및 색상을 지정하는 데 사용되는 약식 속성입니다.
  • border-right-color : border-right-color 속성은 요소의 오른쪽 테두리 색상을 설정하는 데 사용됩니다.
  • border-right-style : border-right-style 속성은 테두리 이미지가 설정되어 있지 않는 한 요소의 오른쪽 테두리 선 스타일(실선, 이중, 대시 등)을 지정하는 데 사용됩니다.
  • border-right-width : border-right-width 속성은 요소의 오른쪽 테두리 너비를 설정하는 데 사용됩니다.
  • border-spacing : border-spacing 속성은 인접한 경계 셀의 경계 사이의 거리를 지정하는 데 사용됩니다.
  • border-style : border-style 속성은 테두리 이미지가 설정되어 있지 않는 한 요소 테두리의 선 스타일(실선, 이중, 대시 등)을 지정하는 데 사용되는 약식 속성입니다.
  • border-top : border-top 속성은 요소 상단 테두리의 너비, 스타일 및 색상을 지정하는 데 사용되는 약식 속성입니다.
  • border-top-color :border-top-color 속성은 요소의 상단 테두리 색상을 설정하는 데 사용됩니다.
  • border-top-left-radius :border-top-left-radius 속성은 요소의 왼쪽 상단 모서리를 둥글게 만드는 데 사용됩니다
  • border-top-right-radius : border-top-right-radius 속성은 요소의 오른쪽 상단 모서리를 둥글게 만드는 데 사용됩니다.
  • border-top-style : border-top-style 속성은 테두리 이미지 세트가 없는 한 요소의 위쪽 테두리 선 스타일(실선, 이중, 대시 등)을 지정하는 데 사용됩니다.
  • border-left-width :border-left-width 속성은 요소의 왼쪽 테두리 너비를 설정하는 데 사용됩니다.
  • bottom :bottom 속성은 위치 지정 컨텍스트와 관련하여 위치 지정 요소의 아래쪽 오프셋을 지정하는 데 사용되는 오프셋 속성입니다.
  • box-shadow : box-shadow 속성은 요소에 하나 이상의 그림자를 드리우는 데 사용됩니다. 실제로 모든 요소는 이 속성을 사용하여 그림자를 얻을 수 있습니다
  • box-sizing : box-sizing 속성은 "상자 모델"과 관련하여 요소의 크기 조정이 발생하는 방식을 변경하는 데 사용됩니다.
  • break-after : break-after 속성은 페이지 나누기, 영역 나누기 또는 열 나누기가 적용된 요소 뒤에 발생해야 하는지 여부를 지정하는 데 사용됩니다.
