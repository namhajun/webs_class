  • mask-border-outset : mask-border-outset 속성은 테두리 마스크 이미지 영역이 요소의 테두리 상자 영역을 넘어 확장되는 양을 지정하는 데 사용됩니다.
  • mask-border-repeat : mask-border-repeat 속성은 테두리 이미지의 측면과 중간 부분에 대한 이미지의 크기를 조정하고 바둑판식으로 배열하는 방법을 지정합니다.
  • mask-border-slice :mask-border-slice 속성은 마스크 테두리 이미지로 사용할 이미지를 9개 조각으로 "slice"하는 데 사용됩니다.
  • mask-border-source :mask-border-source 속성은 마스크 테두리 이미지로 사용할 이미지를 지정하는 데 사용됩니다.
  • mask-border-width : mask-border-width 속성은 mask-border-slice 속성으로 생성된 마스크 테두리 이미지 슬라이스의 크기를 조정하는 데 사용됩니다.
  • mask-clip :mask-clip 속성은 마스크의 영향을 받는 요소의 영역을 결정하는 마스크 페인팅 영역을 결정합니다.
  • mask-composite :mask-composite 속성은 마스크 레이어 이미지가 그 아래(또는 뒤에 있는) 다른 마스크 레이어 이미지와 합성되는 방법을 지정하는 데 사용됩니다
  • mask-image : mask-image 속성은 요소의 마스크 레이어로 사용할 하나 이상의 쉼표로 구분된 이미지를 지정하는 데 사용됩니다.
  • mask-mode : mask-mode 속성은 CSS 마스크 레이어 이미지를 알파 마스크로 처리할지 휘도 마스크로 처리할지 지정하는 데 사용됩니다.
  • mask-origin :mask-origin 속성은 요소의 마스크 위치 지정 영역을 지정합니다.
  • mask-position : mask-position 속성은 마스크 위치 지정 영역 내에서 마스크 레이어 이미지의 초기 위치(크기 조정 후)를 지정합니다.
  • mask-repeat : mask-repeat 속성은 마스크 레이어 이미지가 반복(또는 타일링)되는지 여부와 방법을 지정하는 데 사용됩니다.
  • mask-size :mask-size 속성은 마스크 레이어 이미지의 크기를 지정하는 데 사용됩니다.
  • mask-type : mask-type 속성은 SVG 요소의 내용을 luminance mask 로 처리할지 alpha mask.로 처리할지 지정하는 데 사용됩니다.
  • max-height :max-height 속성은 지정된 요소의 최대 높이를 설정하는 데 사용됩니다
  • max-width : max-width 속성은 지정된 요소의 최대 너비를 설정하는 데 사용됩니다.
  • min-height :min-height 속성은 지정된 요소의 최소 높이를 설정하는 데 사용됩니다.
  • min-width : min-width 속성은 지정된 요소의 최소 너비를 설정하는 데 사용됩니다.
  • mix-blend-mode : mix-blend-mode 속성은 요소를 배경과 혼합하기 위한 혼합 모드를 지정하는 데 사용됩니다.
  • object-fit : object-fit 속성은 대체된 요소의 내용을 사용된 높이와 너비로 설정된 상자에 맞추는 방법을 지정합니다.
  • object-position :object-position 속성은 상자 안에 있는 대체된 요소의 내용 정렬을 결정합니다
